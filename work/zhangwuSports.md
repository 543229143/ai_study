## 一、Skills 功能总览

| Skill 名称 | 描述 | 功能点 |
|-----------|------|--------|
| `osi-new-fundcode` | OSI 机构接入 | 1、告知机构号和机构名，自动完成 11 步全流程：SQL → 分支 → 常量 → 版本号 → 提交 → 编译发布 → 合并分支 → 文档 → Nacos → Mock → 部署<br>2、支持全流程/H5/导下载三种模式<br>3、预检门禁提前发现环境问题<br>4、11 步执行完毕自动回执可追溯 |
| `non-osi-new-fundcode` | 非 OSI 机构接入 | 1、启动/继续/重跑三种指令，中断后可恢复<br>2、全流程自动化：文档 → SQL → 代码生成（GOA/LCS/AMS/LPS）→ Mock → Nacos<br>3、上线前独立提供告警白名单和脱敏配置<br>4、按契约约束代码范围，避免无效生成 |
| `technical-design-doc` | 技术文档生成 | 1、跨 lps/ams/lcs/goa 四仓库扫描代码<br>2、按标准化模板生成技术方案文档<br>3、自动影响面检查，识别兼容性风险、配置遗漏、联调范围<br>4、文档自动落库归档 |
| `feature-development` | 功能开发全流程 | 1、根据技术文档自动完成从预检到代码生成的全流程<br>2、自动创建分支、执行数据库变更、推送 Nacos 配置<br>3、按依赖顺序逐应用生成代码并编译<br>4、全部完成后自动触发 Code Review |
| `alibabacloud-sls-query` | SLS 日志查询 | 1、自然语言描述查询需求，自动执行日志查询<br>2、大时间范围自动分段补齐保证结果完整<br>3、生产环境预配置，无需额外配置 |
| `issue-investigation` | 问题排查 | 1、贴 traceId 或描述问题即可自动排查，可视化呈现链路<br>2、自动拉日志、查数据库、关联源码定位根因<br>3、输出完整的排查链路图和根因结论<br>4、支持 dev/sit 多环境 |
| `goa-existing-org-mask` | 脱敏增量扫描 | 1、发版前自动扫描新增/改动接口字段，与生产脱敏基线比对<br>2、自动归类字段类型生成 Nacos 脱敏配置<br>3、避免人工排查遗漏敏感字段，保障数据合规 |
| `ai-cr-workflow` | AI 代码审查 | 1、5 步闭环：扫描代码 → 标注问题 → 自动修复 → 验证 → 完成<br>2、自动关联飞书需求文档，审查历史可追溯<br>3、问题编号管理，修复补丁自动生成<br>4、标准化输出模板保证报告质量 |
| `api-request` | HTTP 请求（底层能力） | 1、支持多种认证方式和请求格式<br>2、作为底层能力被 Nacos/Mock/部署等技能复用 |
| `ca-sign` | CA 签章接入 | 1、收集文档 → 扫描代码 → 写入文档 → 用户确认后才开发<br>2、确认后自动建分支/改 pom/部署<br>3、端到端对接：签约回调 + 签章推送 + LPS 回调<br>4、存量机构 Mock，无需联调即可自测<br>5、支持联调与收尾闭环 |
| `sql-executor` | SQL 执行器 | 1、支持表结构变更和数据操作，DML 需人工确认<br>2、表结构变更自动联动更新配套代码<br>3、支持多套数据库环境切换，记录执行结果 |
| `code-refactor` | 代码重构方案 | 1、自动生成重构方案并内置灰度策略支持平滑上线<br>2、自检清单覆盖死代码、事务边界等风险<br>3、多轮提案→反驳→裁决机制收敛方案降低风险 |
| `dev-auto-deploy` | Dev 自动部署 | 1、一键完成版本检查、编译部署、运维部署<br>2、多项目并行部署节省等待时间 |
| `goa-existing-org-mock` | GOA 存量机构 Mock | 1、自动识别 OSI/非 OSI 机构类型选择不同策略<br>2、仅需 3 个指令（生成/上传/重跑）<br>3、复核后才上架，未复核禁止上传<br>4、按接口路径自动去重 |
| `auto-merge-preview` | 自动创建 MR | 1、自动分析变更内容生成结构化描述<br>2、支持多项目同时创建 MR<br>3、创建后企微自动通知附带 MR 链接 |
| `comment-review-rewrite-zh` | 中文注释检查 | 1、全链路检查注释与代码一致性，跨项目核查<br>2、删除过期误导注释，统一术语风格<br>3、为复杂逻辑补充中文注释 |
| `api-dubbo-docs-generator` | Dubbo 接口文档 | 1、自动扫描服务接口，按方法粒度生成文档<br>2、文档包含接口名称、路径、参数、返回值 |

## 二、基建成果

| 模块 | 功能点 |
|------|--------|
| 无用代码清理 | 跨项目识别并清理死代码，自动验证确保不影响线上功能，降低维护成本 |
| 核心业务重构 | ams 账户开户/更新流程重组、借款试算、借款申请核心逻辑重构优化，提升代码可维护性 |
| 告警降噪 | 通过重复告警去重、漏告警补充、误告警过滤等多手段优化，告警降噪效果显著 |
| CAT 监控配置 | 搭建自动化巡检覆盖账务/授信/运营/买量 4 个业务组共 19 个服务，分级阈值控制告警量，企微自动通知 |
| 远程 Skill 同步 | 技能统一托管全局同步，一次发布全员生效，新增/更新/删除无需逐项目维护，新项目零配置直接使用 |
